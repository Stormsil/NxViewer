# Suggested Commands (Windows / PowerShell)

## Build
```powershell
dotnet build .\NxTiler.sln -c Debug
```

## Test
```powershell
dotnet test .\NxTiler.sln -c Debug
```

## Run (App)
```powershell
dotnet run --project .\src\NxTiler.App\NxTiler.App.csproj -c Debug
```

## Clean
```powershell
dotnet clean .\NxTiler.sln
```
